### **接口描述**
获取模型版本列表。

### **权限说明**
鉴权认证机制的详细内容请参见[鉴权认证](https://cloud.baidu.com/doc/Reference/s/Njwvz1wot)。

### **请求结构**
```bash
GET ?action=DescribeModelVersions&modelId=xxx&pageNumber=xxx&pageSize=xxx
HTTP/1.1
Host: aihc.bj.baidubce.com
Authorization: authorization string
version: v2
```
### **请求头域**
除公共头域外，无其它特殊头域。

### **请求参数**
|参数名称|类型|是否必选|参数位置|参数描述|
|-|-|-|-|-|
|modelId|String|是|Query参数|模型ID|
|pageNumber|Integer|否|Query参数|分页参数，没传默认1|
|pageSize|Integer|否|Query参数|分页大小，没传默认返回全部|

### **响应头域**
除公共头域外，无其它特殊头域。

### **响应参数**
|参数名称|类型|描述|
|-|-|-|
|totalCount|Integer|模型版本总数|
|versions|List<[ModelVersionEntry](AIHC/API参考/数据结构.md#ModelVersionEntry)\>|模型版本列表|

### **请求示例**
无。
