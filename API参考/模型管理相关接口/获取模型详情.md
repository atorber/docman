### **接口描述**
获取模型详情。

### **权限说明**
鉴权认证机制的详细内容请参见[鉴权认证](https://cloud.baidu.com/doc/Reference/s/Njwvz1wot)。

### **请求结构**
```bash
GET ?action=DescribeModel&modelId=xxx
HTTP/1.1
Host: aihc.bj.baidubce.com
Authorization: authorization string
version: v2
```
### **请求头域**
除公共头域外，无其它特殊头域。

### **请求参数**
|参数名称|类型|是否必选|参数位置|参数描述|
|-|-|-|-|-|
|modelId|String|是|Query参数|模型ID|

### **响应头域**
除公共头域外，无其它特殊头域。

### **响应参数**
|参数名称|类型|描述|
|-|-|-|
|name|String|模型名称|
|id|String|模型ID|
|initSource|String|模型创建时的来源UserUpload：用户上传|
|modelFormat|String|模型格式|
|description|String|描述|
|createdAt|String|创建时间|
|updatedAt|String|更新时间|
|owner|String|所有者|
|ownerName|String|所有者名称|
|visibilityScope|String|可见范围|
|versionEntry|[ModelVersionEntry](AIHC/API参考/数据结构.md#ModelVersionEntry)|最新版本的详细信息|

### **请求示例**
无。
