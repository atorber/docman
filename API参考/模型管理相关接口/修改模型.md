### **接口描述**
修改模型。

### **权限说明**
鉴权认证机制的详细内容请参见[鉴权认证](https://cloud.baidu.com/doc/Reference/s/Njwvz1wot)。

### **请求结构**
```bash
POST ?action=ModifyModel&modelId=xxx
HTTP/1.1
Host: aihc.bj.baidubce.com
Authorization: authorization string
version: v2
```
### **请求头域**
除公共头域外，无其它特殊头域。

### **请求参数**
|参数名称|类型|是否必选|参数位置|参数描述|
|-|-|-|-|-|
|modelId|String|是|Query参数|模型ID|
|name|String|是|Body参数|模型名称|
|description|String|是|Body参数|描述|

### **响应头域**
除公共头域外，无其它特殊头域。

### **响应参数**
无响应参数。

### **请求示例**
```json
{
    "name": "my-model-1",
    "description": "my model"
}
```
### **响应示例**
```json
{
    "requestId": "384ccc3f-fb52-4bdc-be37-978a0169bbed",
    "id": "mo-vnufEepi"
}
```
