### **接口描述**
创建数据集版本。

### **权限说明**
鉴权认证机制的详细内容请参见[鉴权认证](https://cloud.baidu.com/doc/Reference/s/Njwvz1wot)。

### **请求结构**
```bash
POST ?action=CreateDatasetVersion&datasetId=xxx
HTTP/1.1
Host: aihc.bj.baidubce.com
Authorization: authorization string
version: v2
```
### **请求头域**
除公共头域外，无其它特殊头域。

### **请求参数**
|参数名称|类型|是否必选|参数位置|参数描述|
|-|-|-|-|-|
|datasetId|String|是|Query参数|数据集ID|
|description|String|否|Body参数|版本描述|
|storagePath|String|是|Body参数|存储路径|
|mountPath|String|是|Body参数|默认挂载路径|

### **响应头域**
除公共头域外，无其它特殊头域。

### **响应参数**
|参数名称|类型|描述|
|-|-|-|
|id|String|数据集版本ID|

### **请求示例**
```json
{
    "description": "new version",
    "storagePath": "/path/to/version2",
    "mountPath": "/mnt/datasets/my-dataset-1/v2"
}
```
### **响应示例**
```json
{
    "requestId": "2642efe1-12ec-4ad5-9e34-7911e3430041",
    "id": "ds-1Wi75n2k"
}
```
