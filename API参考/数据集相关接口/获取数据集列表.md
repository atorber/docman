### **接口描述**
获取数据集列表。

### **权限说明**
鉴权认证机制的详细内容请参见[鉴权认证](https://cloud.baidu.com/doc/Reference/s/Njwvz1wot)。

### **请求结构**
```bash
GET ?action=DescribeDatasets&keyword=xxx&storageType=xxx&storageInstances=xxx&importFormat=xxx&pageNumber=xxx&pageSize=xxx
HTTP/1.1
Host: aihc.bj.baidubce.com
Authorization: authorization string
version: v2
```
### **请求头域**
除公共头域外，无其它特殊头域。

### **请求参数**
|参数名称|类型|是否必选|参数位置|参数描述|
|-|-|-|-|-|
|keyword|String|否|Query参数|名称关键字|
|storageType|String|否|Query参数|存储类型PFS：并行存储PFSBOS：对象存储BOS|
|storageInstances|String|否|Query参数|存储实例列表，英文逗号(,)分隔|
|importFormat|String|否|Query参数|导入格式FILE：文件FOLDER：文件夹|
|pageNumber|Integer|否|Query参数|请求分页参数，表示第几页，默认值为1|
|pageSize|Integer|否|Query参数|单页结果数，不传递该参数默认返回全部|

### **响应头域**
除公共头域外，无其它特殊头域。

### **响应参数**
|参数名称|类型|描述|
|-|-|-|
|totalCount|Integer|数据集总数|
|datasets|List<[Dataset](AIHC/API参考/数据结构.md#Dataset)\>|数据集列表|

### **请求示例**
无。
