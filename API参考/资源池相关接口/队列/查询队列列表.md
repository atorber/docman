### 描述
获取指定资源池的队列列表

* 支持指定关键字查询
* 支持分页

### 请求结构
```
GET ?action=DescribeQueues&resourcePoolId=xxxx
Host:aihc.bj.baidubce.com
Authorization:authorization string
ContentType: application/json
version: v2    
```
### 请求头域
除公共头域外，无其它特殊头域。

### 请求参数
|参数名称|类型|是否必须|参数位置|说明|
|-|-|-|-|-|
|resourcePoolId|String|是|Query 参数|标识资源池的唯一标识符|
|keywordType|String|否|Query 参数|筛选关键字类型，当前仅支持queueName和queueId|
|keyword|String|否|Query 参数|关键字值|
|pageNumber|Number|否|Query 参数|请求分页参数，表示第几页|
|pageSize|Number|否|Query 参数|单页结果数，默认值为10|

### 返回头域
除公共头域外，无其他特殊头域。

### 返回参数
|参数名称|类型|说明|
|-|-|-|
|keywordType|String|筛选关键字类型，当前仅支持queueName和queueId|
|keyword|String|关键字值|
|pageNumber|Number|请求分页参数，表示第几页，默认值为1|
|pageSize|Number|单页结果数，默认值为10|
|totalCount|Number|队列总数|
|queues|List<QueueItem>​|队列详情列表|
