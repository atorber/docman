### 描述
创建队列，目前支持创建托管资源队列。

### 请求结构
```
POST ?action=CreateQueue
Host:aihc.bj.baidubce.com
Authorization:authorization string
ContentType: application/json
version: v2
```
### 请求头域
除公共头域外，无其它特殊头域。

### 请求参数
|参数名称|类型|是否必须|说明|
|-|-|-|-|
|queueName|String|是|队列名称：首尾只能是小写字母或数字，支持数字、小写字母和-，最长 63 个字符|
|description|String|否|队列描述信息：最长 300 个字符|
|resourcePoolId|String|是|队列所属资源池ID|
|opened|Boolean|否|队列是否打开：默认为 false|
|queueingStrategy|String|是|队列调度策略：支持BestEffortFIFO、StrictFIFO|
|requeueTimeout|Number|否|超时重排阈值：1、StrictFIFO 队列策略不支持超时重排，2、超时重排阈值，单位分钟。空或 0 代表关闭超时重排。取值范围 5～6000|
|bindingNodes|List<[BindingNodeInfo](AIHC/API参考/数据结构.md#BindingNodeInfo)>|否|节点绑定信息：不指定时创建没有节点的空队列|

### 返回头域
除公共头域外，无其他特殊头域。

### 返回参数
|参数名称|类型|说明|
|-|-|-|
|queueName|String|创建的队列名称|
|queueId|String|创建的队列ID|

### 请求示例
```
{
    "queueName": "openapi-create-01",
    "queueingStrategy": "BestEffortFIFO",
    "description": "openapi create test",
    "requeueTimeout": 5,
    "bindingNodes": [
        {
            "machineSpec": "aihc.gn5.c16m64.1a10",
            "count": 1,
            "zoneName": "cn-bj-d"
        }
    ],
    "opened": true,
    "resourcePoolId": "aihc-kis89e33o9hu"
}
```

### 返回示例
```
{
    "queueId": "aihcq-8yoowbfhvmf3",
    "queueName": "openapi-create-01"
}
```
