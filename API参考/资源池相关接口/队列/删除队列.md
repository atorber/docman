  
### 描述
删除队列，目前支持删除托管资源队列。

### 请求结构
```
POST ?action=DeleteQueue&queueId=xxx
Host:aihc.bj.baidubce.com
Authorization:authorization string
ContentType: application/json
version: v2
```
### 请求头域
除公共头域外，无其它特殊头域。

### 请求参数
|参数名称|类型|是否必须|说明|
|-|-|-|-|
|queueId|String|是|队列ID|

### 返回头域
除公共头域外，无其他特殊头域。

### 返回参数
无

### 返回示例
无
