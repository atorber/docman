  
### 描述
创建节点，目前支持创建托管资源节点。

### 请求结构
```
POST ?action=CreateNode
Host:aihc.bj.baidubce.com
Authorization:authorization string
ContentType: application/json
version: v2
```
### 请求头域
除公共头域外，无其它特殊头域。

### 请求参数
|参数名称|类型|是否必须|说明|
|-|-|-|-|
|nodeSet|List<[NodeSet](AIHC/API参考/数据结构.md#NodeSet)>|是|创建的节点规格，只支持创建一种规格|

### 返回头域
除公共头域外，无其他特殊头域。

### 返回参数
|参数名称|类型|说明|
|-|-|-|
|result|List<String>|创建的节点列表ID|
|requestId|String|请求ID|
|code|Number|返回码|
|message|String|返回信息|

### 请求示例

```
{
  "nodeSet": [
    {
      "addNodeSpec": {
        "zoneName": "cn-bj-d",
        "machineSpec": "aihc.g5.c2m8",
        "resourceChargingOption": {
          "chargingType": "Prepaid", // Postpaid
          "purchaseTime": 1,
          "purchaseTimeUnit": "MONTH",
          "autoRenew": true,
          "autoRenewTime": 1,
          "autoRenewTimeUnit": "MONTH"
        },
        "networkInfo": {
          "subnetIds": [
            "sbn-0j2yiz8syt8p"
          ],
          "subnetCidr": "10.5.0.0/16"
        }
        // "ehcClusterId": ""
      },
      "count": 2
    }
  ]
}
```
