### 描述
删除节点，目前支持删除托管资源节点。

### 请求结构
```
POST ?action=DeleteNode&resourcePoolId=xxx
Host:aihc.bj.baidubce.com
Authorization:authorization string
ContentType: application/json
version: v2
```
### 请求头域
除公共头域外，无其它特殊头域。

### 请求参数
|参数名称|类型|是否必须|说明|
|-|-|-|-|
|resourcePoolId|String|否|资源池ID|
|instanceIds|List<[String]()>|是|实例ID列表|

### 返回头域
除公共头域外，无其他特殊头域。

### 返回参数
|参数名称|类型|说明|
|-|-|-|
|code|Number|返回状态码|
|message|String|返回信息|
|requestId|String|请求ID|

### 请求示例
```
{
    "instanceIds": [
        "aihcn-9idn0r5m1puz"
    ]
}
```

### 返回示例
```
{
    "code": 200,
    "message": "success",
    "requestId": "3c4f5a3e-1aaa-4b1a-8dda-6f913d8246c7",
}
```
