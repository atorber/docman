### 描述
删除资源池，目前支持删除托管资源池。
需要先清除资源池中的队列，释放节点后才能删除资源池。

### 请求结构
```
POST ?action=DeleteResourcePool&resourcePoolId=xxxx
Host:aihc.bj.baidubce.com
Authorization:authorization string
ContentType: application/json
version: v2    
```
### 请求头域
除公共头域外，无其它特殊头域。

### 请求参数
|参数名称|类型|是否必须|说明|
|-|-|-|-|
|resourcePoolId|String|是|资源池 ID|

### 返回头域
除公共头域外，无其他特殊头域。

### 返回参数
|参数名称|类型|说明|
|-|-|-|
|resourcePoolId|String|删除的资源池 ID|
|requestId|String|删除请求的requestId|

### 返回示例
```
{
    "requestId": "66f82a87-7c0a-446d-a42b-a8cbf7302525",
    "resourcePoolId": "aihc-dk35w6cni277"
}
```
