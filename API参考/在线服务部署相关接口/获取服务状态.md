### 接口描述
获取服务状态。

### 请求结构
```bash
GET ?action=DescribeServiceStatus&serviceId=serviceId
HTTP/1.1
Host:aihc.bj.baidubce.com
version: v2
Authorization:authorization string 
```
### 请求头域
除公共头域外，无其它特殊头域。

### 请求参数
|参数名|参数类型|是否必须|参数位置|参数说明|
|-|-|-|-|-|
|serviceId|String|是|Query参数|单个服务id|

### 返回头域
除公共头域，无其它特殊头域。

### 返回参数
|参数名|参数类型|参数说明|
|-|-|-|
|service|Map<String, [ServiceBriefStat](AIHC/API参考/数据结构.md#ServiceBriefStat)>|服务的状态，map的key对应serviceId，value为应用的状态信息|
|requestId|String|唯一标识|

### 请求示例
```bash
GET ?action=DescribeServiceStatus&serviceId=s-r7d45dcffa0a
HTTP/1.1
Host:aihc.bj.baidubce.com

```
