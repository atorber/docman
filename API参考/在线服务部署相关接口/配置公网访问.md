### 接口描述
配置公网访问

### 请求结构
```bash
POST ?action=ModifyServiceNetConfig&publicAccess=publicAccess&eip=eip
HTTP/1.1
Host:aihc.bj.baidubce.com
version: v2
Authorization:authorization string 
```
### 请求头域
除公共头域外，无其它特殊头域。

### 请求参数
|参数名|参数类型|是否必须|参数位置|参数说明|
|-|-|-|-|-|
|serviceId|String|是|Query参数|服务ID|
|publicAccess|Boolean|是|Query参数|是否开启公网，ture开启，false关闭|
|eip|String|否|Query参数|未使用云原生网关（networkType=""）开启公网时必填，填入未绑定的EIP IP，如需更换EIP，需要先关闭公网，再选新的EIP。开启云原生网关（networkType=aiGateway）自动分配eip，指定不生效|

### 返回头域
除公共头域，无其它特殊头域。

### 返回参数
|参数名称|类型|说明|
|-|-|-|
|requestId|String|请求唯一标识|
