# 查询推理服务日志
## 描述
获取推理服务中实例的日志。

## 请求结构
```
POST ?action=DescribeServiceLogs
Host:aihc.bj.baidubce.com
Authorization:authorization string
ContentType: application/json
X-API-Version: v2    
```
## 请求头域
除公共头域外，无其它特殊头域。

## 请求参数
|参数名称|类型|是否必须|参数位置|说明|
|-|-|-|-|-|
|serviceId|String|是|Body 参数|服务ID|
|instanceId|String|是|Body 参数|实例ID|
|startTime|String|否|Body 参数|日志的起始时间，RFC3339格式；例如：2026-03-20T01:40:35Z。|
|endTime|String|否|Body 参数|日志的结束时间，RFC3339格式；例如：2026-03-27T01:40:35Z。。|
|maxLines|Integer|否|Body 参数|日志的最大行数；未设置和超过100行则返回100行日志。|
|marker|String|否|Body 参数|日志查询标识符，配合返回值中的 nextMarker 参数使用。使用方法为：第一次请求不写 marker 参数，获取返回值，如果 nextMarker 字段不为空，则将nextMarker 的值作为 marker 参数传入，获取更多的日志，直到 nextMarker 为空|

## 返回头域
除公共头域，无其它特殊头域。

## 返回参数
|参数名称|类型|说明|
|-|-|-|
|requestId|String|请求ID|
|logs|List<String>|日志条目|
|nextMarker|String|下次查询的日志标识符|

## 请求示例
```
{
    "serviceId": "s-re4267be4f9b",
    "instanceId": "s-re4267be4f9b-7cb86bcd7f-wm5f5", #实例列表中实例ID
    "startTime": "2026-02-11T03:42:51Z",
    "endTime": "2026-02-11T09:20:32Z",
    "maxLines": 100
}
```
