
### 接口描述
查询开发机详情。

### 请求结构
```
GET ?action=DescribeDevInstance&devInstanceId=dv-343423e3
HTTP/1.1
Host:aihc.bj.baidubce.com
version: v2
Authorization:authorization string
```
### 请求头域
除公共头域外，无其它特殊头域。

### 请求参数
|参数名|参数类型|是否必须|参数位置|参数说明|
|-|-|-|-|-|
|devInstanceId|String|是|Query参数|操作对象|

### 返回头域
除公共头域，无其它特殊头域。

### 返回参数
|参数名称|类型|说明|
|-|-|-|
|requestId|string|请求唯一标识|
|devInstance|[DevInstanceDetail](https://cloud.baidu.com/doc/AIHC/s/Imb0gupts#devinstancedetail)|开发机详情|
