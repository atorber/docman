  
开发机是百舸平台为开发者提供的在线编译、调试代码和模型开发的模块。开发机支持使用 WebIDE 在线开发，或通过 SSH 连接开发机远程开发，也提供了持久化的共享文件系统用于存储开发中的数据。开发机在关机后，平台会释放开发机的算力，通过备份开发机可以保存之前的操作、下载的数据和配置环境等。

开发机创建在已有百舸资源池的指定队列中，**与训练任务、推理服务共享资源池GPU/CPU/内存等资源，需根据开发需求合理分配资源，以免对训推任务造成影响**。

1. 登录百舸异构计算平台控制台，点击左侧导航栏中【模型开发/开发机】进入列表页面
2. 点击列表页面左上方的【+创建实例】进入创建页面
3. 在创建页面填写相关参数，具体参数如下

| 字段 | 说明 |
| --- | --- |
| 实例名称 | **必填，**填写开发机的名称<br>支持小写字母、数字以及 - 且开头必须是小写字母，结尾必须是小写字母或数字，长度1-50 |
| 资源池 | **必填，**选择开发部署的百舸资源池 |
| 队列 | **必填，**选择创建开发机的队列 |
| 镜像地址 | **必填，**选择开发机的镜像<br>支持百舸镜像、自定义镜像（私有的镜像仓库需要填写用户名和密钥，企业版CCR镜像地址默认为VPC地址，需要在[企业版CCR/指定实例详情页/访问控制/私有网络/添加私有网络](https://console.bce.baidu.com/ccr/#/ccr/company/instance/list)中配置VPC访问控制策略，添加资源池所在VPC<br>注意：当前自定义镜像仅支持通过【制作镜像】功能备份的镜像，不支持用户自定义镜像，因此首次创建开发机时需选择【百舸镜像】创建开发机 |
| 环境变量 | **选填，**启动开发机时注入的环境变量，开发过程中使用的密钥等敏感信息可以设置为环境变量 |
| 加速芯片申请 | **必填，**默认不申请，启用时需进一步选择加速芯片类型、加速芯片数 |
| 加速芯片类型 | **必填，**选择加速芯片类型 |
| 加速芯片数 | **选填，**设置加速芯片数，不填写或填写0时表示不带加速芯片 |
| CPU | **选填**，默认值：不限；设置开发机占用的最多CPU核数 |
| 内存 | **选填**，默认值：不限；设置开发机占用的最大内存大小 |
| 共享内存 | **选填**，默认值10Gi，一般使用默认值即可，如业务有额外需求，可以按需修改<br>共享内存（shared memory）用于Linux中不同进程之间数据交换和共享，以提高应用程序的性能和效率。 |
| 存储挂载 | **必填**，支持挂载PFS、对象存储Bos；挂载Bos对象存储时，需要提供AK/SK完成Bos授权。 |
| 开启SSH | **选填**，默认值：不开启；不开启时默认不显示SSH公钥、开启公网选项<br>前置条件：开启SSH时，系统会自动为用户在资源池中创建一个BLB |
| SSH公钥 | **必填**，开发机 SSH 登录时鉴权公钥；处于安全考虑，SSH登录仅支持公钥认证，不支持用户名 + 密码的认证方式。<br>通常填写发起 SSH 登录机器上的`~/.ssh/id_rsa.pub`文件内容即可。若该文件不存在，可用 `ssh-keygen`命令生成。发起 SSH 登录时，请注意检查私钥是否配对。<br>如需从多台机器发起 SSH 登录可填写多个公钥，也可直接编辑开发机`~/.ssh/authorized_keys`文件内的公钥列表。 |
| SSH监听端口 | **必填**，开发机容器SSH服务端口号，默认为22，需要自定义服务端口号时填写SSH服务的容器的实际端口号 |
| SSH端口号 | **必填**，从内网或公网连接开发机实例的端口，默认支持在子网环境登录SSH，若需要通过公网登录，可以打开【开启公网开关】，需在[BLB实例/安全组](https://console.bce.baidu.com/blb/#/appblb/list)设置中放通配置的端口,开发机创建的BLB实例名称以“dev-”开头，例如dev-47e7dcc4** |
| 开启公网 | **选填**，默认值：不开启；<br>支持两种方式登录，分别为公网登录和私网登录<br>公网登录：在任意连通公网的机器上，均能使用 SSH 公网指令登录开发机，开发机详情页可以查看公网连接地址。<br>内网登录：通过内网登录开发机，需确保客户端机器能够连接BLB内网，内网地址为VPC地址，能够访问VPC的机器均可连接开发机。 |
| WebIDE服务监听端口 | **必填**，开发机容器的WebIDE服务端口，预计镜像默认为8600，自定义镜像时需填写容器内服务的实际启动端口 |
