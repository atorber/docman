  
## 开发机生命周期管理
开发机的的生命周期指从创建开始到删除结束，开发机实例所经历的状态。在不同状态，开发机的登录、操作不同。以下介绍开发机状态说明及状态转换说明：

| 状态 | 描述信息 |
| --- | --- |
| 创建中 | 创建开发机的配置信息 |
| 排队中 | 开发机正在排队流程中，若资源不足会一直处于排队状态 |
| 部署中 | 调度成功开始部署 |
| 运行中 | 开发机已创建完成，可以通过 WebIDE或 SSH 连接开发机 |
| 停止实例中 | 开发机正在停止当前运行实例中 |
| 已停止实例 | 开发机成功停止当前实例，停止实例后不会丢失之前的操作和环境 |
| 开启实例中 | 开发机正在开启当前实例 |
| 更新中 | 开发机当先实例正在更新 |
| 扩容中 | 开发机当前实例正在扩容 |
| 制作镜像中 | 正在保存当前开发机环境 |
| 删除中 | 正在删除开发机 |
| 重启中 | 正在重新启动当前开发机 |
| 已释放 | 已经释放当前开发机资源 |
| 已失败 | 拉取镜像失败，内部请求参数异常 |
| 异常 | 开发机异常，无法正常使用，可在状态后查看异常原因。 |


## 变更配置
如需修改开发机的配置，请点击【变更配置】可修改开发机的资源配置、存储挂载、访问端口、访问权限等信息。

## 开发机停止实例
若开发机暂时不再使用，可以停止当前实例，来关闭开发机。开发机停止实例后，会保存环境配置、数据等信息。但会释放开发机占用的算力资源，再次开机可能会出现算力资源不足排队等待的情况。

![image \(1\).png](https://bce.bdstatic.com/doc/bce-doc/AIHC/image%20%281%29_8587479.png)


## 开发机制作镜像
如需基于当前现有的开发机实例快速创建一个相同镜像资源，可以使用制作镜像功能。

![image.png](https://bce.bdstatic.com/doc/bce-doc/AIHC/image_96b7c0c.png)

## 开发机免释放重启
在资源池详情中设置免释放重启，开发机实例将被重启，但重启过程中不会释放开发机资源，无须等待排队。

![image.png](https://bce.bdstatic.com/doc/bce-doc/AIHC/image_5d70b69.png)

## 开发机定时停止实例
在资源池详情中设置定时停止实例策略，支持自定义时长，最低时长不得低于1小时，仅支持整时数，例如1.5小时则视为2小时。

![image.png](https://bce.bdstatic.com/doc/bce-doc/AIHC/image_527a956.png)

## 开发机扩容
如需要对当前开发机扩容云磁盘，可以使用扩容功能，在开发机实例详情扩容云磁盘。

![image.png](https://bce.bdstatic.com/doc/bce-doc/AIHC/image_d2f6d34.png)

![image.png](https://bce.bdstatic.com/doc/bce-doc/AIHC/image_485cc8d.png)


## 复制开发机
如需基于当前现有的开发机快速创建一个新开发机，可以使用复制实例功能。
![image.png](https://bce.bdstatic.com/doc/bce-doc/AIHC/image_e5e0523.png)

## 删除开发机
若开发机使用已经结束，可以删除开发机。开发机删除后，会释放开发机的算力。如需保存开发环境和数据，请先制作镜像保存。
![image.png](https://bce.bdstatic.com/doc/bce-doc/AIHC/image_a069a8b.png)
