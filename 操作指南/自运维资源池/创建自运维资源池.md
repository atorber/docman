
## 概述

本文将介绍如何创建自运维计算资源池。

**自运维计算资源池**是百舸内部管理资源的基础单元，一个资源池对应一个CCE集群，一个PFS实例和一个CPROM实例。

在资源池中，用户可以创建多个队列，从而在不同的队列中处理不同业务的工作任务。通常情况下，企业内部一个独立的团队或者项目建议创建一个独立的资源池，实现团队间或项目间的资源隔离。

百舸平台支持用户基于实际使用场景创建具备不同规格GPU套餐的资源池，详细GPU套餐规格请见：[百舸平台支持的GPU套餐](AIHC/产品描述/使用限制.md)。

## 操作步骤
1. 登录[百舸AI计算平台AIHC控制台](https://console.bce.baidu.com/aihc/resources)。 
2. 在左侧菜单栏选择资源池，进入资源池列表页面，点击**创建自运维计算资源**。  

![image.png](https://bce.bdstatic.com/doc/bce-doc/AIHC/image_1d50693.png)

3. 在购买资源池页面，根据实际需求完成相关参数配置。
*  基本信息
![image.png](https://bce.bdstatic.com/doc/bce-doc/AIHC/image_1cf19e6.png)

| 配置项 | 必填/选填 | 配置说明 |
| --- | --- | --- |
| 资源池名称 | 必填 | 输入资源池名称，支持大小写字母、数字、中文以及-_ /.特殊字符，必须以字母开头，长度1-65。 |
| 当前地域 | 必填 | 选择集群所在的物理区域。 |
| 可用区 | 必填 | 该可用区中包括您所需的VPC、节点配置/类型、存储实例和监控大盘实例。 |

* 网络配置
![image.png](https://bce.bdstatic.com/doc/bce-doc/AIHC/image_5e5d9d1.png)

| 配置项 | 必填/选填 | 配置说明 |
| --- | --- | --- |
| 资源池网络 | 必填 | 选择集群中节点网络所在的VPC，您可以在当前可用区新建VPC，每个可用区均已创建默认私有网络。 |
| 子网 | 必填 | 选择集群中节点网络所在的子网，您可以在当前可用区新建子网，每个可用区均已创建默认私有网络。|
| APIServer公网访问 | 必填 | 开启后将为集群API Server绑定公网IP。 |
| 容器网络模式 | 选填 | 默认支持VPC-ENI网络模式。 |
| 容器子网 | 必填 | 请您务必在选择网段前，合理有效地规划集群的网络，避免出现网络冲突，否则可能存在风险。[CCE集群网络说明及规划](https://cloud.baidu.com/doc/CCE/s/Mjx9smuum) |
| Cluster IP网段 | 必填 | 配置当前集群的ClusterIP网段 |
| 安全组 | 必填 | 用户可使用普通安全组或企业安全组 |

* 节点配置
![image.png](https://bce.bdstatic.com/doc/bce-doc/AIHC/image_26aa5cf.png)

| 配置项 | 必填/选填 | 配置说明 |
| --- | --- | --- |
| 节点来源 | 必填 | 支持新建节点或直接添加已有服务器资源（已有HPAS节点仅能在创建资源池完成后才能添加）。 |
|付费方式|必填|预付费：先付费后使用服务。您可在账户余额中预先充值，通过账户余额支付购买；也可直接通过银行卡、百度钱包等支付购买。后付费：先使用服务后付费。根据您的使用情况，按分钟实时计费并扣费。因此为保证服务的正常使用，请在使用后付费支付方式的产品前向账户中预先充值。|
|购买时长|必填|选择购买节点的时间|
|自动续费|必填|若您已选购预付费服务，为节省您的操作时间，避免因续费失败造成的服务停服给您带来的不便，百度智能云建议您开通自动续费业务。开通自动续费后，百度智能云将在实例到期前7/5/3/1/0天定时进行自动扣款续费，续费成功或失败都将向您发送短信和邮件提醒。|
|续费周期|必填|开通自动续费后，选择您需要的自动续费周期（按年或按月）与续费周期数量（N个月或N年），点击“确定”。|
|节点选择|必填|根据不同的业务场景，为用户提供了多种不同规格的GPU节点配置/类型。|
|节点数量|必填|当前最少支持2个节点。|
|登陆凭证|必填|支持多种登陆方式，SSH密钥对是一种比常规密码更安全的登录云服务器的方式，更多内容详见[帮助文档\>](https://cloud.baidu.com/doc/BCC/s/5lkqskwd5)|
|可用性检查|必填|开启后，将在资源池创建成功后自动进行可用性测试，以保障集群资源池&节点的运行状态和任务提交效果。|

* 存储配置

![image.png](https://bce.bdstatic.com/doc/bce-doc/AIHC/image_1e920e0.png)

| 配置项 | 必填/选填 | 配置说明 |
| --- | --- | --- |
| 存储实例类型| 选填 | 选择当前VPC内子网下已有的PFS存储实例与集群进行关联。 |
| PFS L2挂载服务 | 选填 | 选择当前VPC内子网下已有的PFS存储实例与集群进行关联。 |


4. 确认资源池配置信息，然后单击提交。下单成功后，您可以在**自运维资源池列表**页面点击**资源池名称**查看详情。
