Ray CPU 类型镜像是 Ray 官方提供的通用运行环境，面向不依赖 GPU 的分布式计算与开发场景。该镜像基于精简的 Linux 系统构建，内置完整的 Ray 运行时和 Python 环境，能够在无需额外硬件或驱动支持的情况下快速启动分布式任务。

该类型镜像具有良好的兼容性和易用性，适合在本地开发机、云服务器以及 Kubernetes 普通节点中部署使用。由于不包含 CUDA、cuDNN 等 GPU 相关组件，其镜像体积相对较小，启动速度更快，资源占用更低，非常适合作为开发调试环境或轻量级生产任务的执行基础。

在功能上，CPU 镜像支持 Ray 的核心能力，包括任务调度、Actor 模型、分布式数据处理（Ray Data）以及基础的模型服务能力（Ray Serve）。对于中小规模的数据处理、并行计算任务或不依赖 GPU 的机器学习工作负载，该镜像可以提供稳定可靠的运行环境。
