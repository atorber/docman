## 概述
托管资源池资源队列是算力资源分配调度的单元，所有使用托管资源池算力的工作负载都需要提交到对应的资源队列中才能获得算力分配。本文将介绍如何创建和管理资源队列。  

## 创建资源队列
1. 登录[百舸异构计算平台AIHC控制台](https://console.bce.baidu.com/aihc/resources)。  
2. 在左侧菜单栏选择托管资源池，进入托管资源池页面，选择**资源队列**，点击**新建资源队列**。  
![image.png](https://bce.bdstatic.com/doc/bce-doc/AIHC/image_85dbc24.png)
3. 在创建资源队列页面，根据实际需求完成相关参数配置。

| 配置项 | 必填/选填 | 配置说明 |
| --- | --- | --- |
| 队列名称 | 必填 | 资源队列名称，支持大小写字母、数字、中文以及-_ /.特殊字符，必须以字母开头，长度1-65。 |
| 备注 | 选填 | 按需填写 |
| 来源类型 | 必填 | 当前仅支持**托管资源池**一种来源。 |
| 资源池 | 必填 | 选择要添加到队列中的节点所属的资源池。 |
| 节点规格 | 选填 | 指定资源池中已有的节点加入队列，也可在队列创建完成后添加节点。 |
| 调度策略 | 必填 | 工作负载调度策略，目前支持**遍历策略**和**FIFO**策略2种。 `遍历策略：当队列中的第一个任务无法出队，则会跳过该任务，然后依次尝试对后续队列中的任务进行出队操作。Fifo策略：当队列中的第一个任务无法出队，系统将反复尝试对第一个任务进行出队操作，而不会跳过。`|

![image.png](https://bce.bdstatic.com/doc/bce-doc/AIHC/image_6de3145.png)

4. 确认配置信息，然后单击提交。创建成功后，您可以在**资源队列**页面查看已创建的资源队列列表。


## 查看队列详情
1. 登录[百舸异构计算平台AIHC控制台](https://console.bce.baidu.com/aihc/resources)。  
2. 在左侧菜单栏选择托管资源池，进入托管资源池页面，选择**资源队列**，点击要查看的**资源队列**名称。  
![image.png](https://bce.bdstatic.com/doc/bce-doc/AIHC/image_02e1fa5.png)
3. 查看当前队列的基本信息。
![image.png](https://bce.bdstatic.com/doc/bce-doc/AIHC/image_678f314.png)

## 节点管理
### 添加节点
1. 登录[百舸异构计算平台AIHC控制台](https://console.bce.baidu.com/aihc/resources)。  
2. 进入资源队列详情，选择**节点管理**，点击**添加节点**。  
![image.png](https://bce.bdstatic.com/doc/bce-doc/AIHC/image_ad3a3ef.png)
3. 在添加节点页面，根据实际需求完成相关参数配置。

### 封锁节点
1. 登录[百舸异构计算平台AIHC控制台](https://console.bce.baidu.com/aihc/resources)。  
2. 进入资源队列详情，选择**节点管理**，找到要封锁的节点，点击操作栏中的“封锁设置”。  
![image.png](https://bce.bdstatic.com/doc/bce-doc/AIHC/image_9bbf999.png)
3. 在封锁设置弹窗中，可按需对节点进行封锁和解封锁操作。开启封锁节点后，节点处于不可调节状态，新状态的Pod将不会调节到该节点上。
![image.png](https://bce.bdstatic.com/doc/bce-doc/AIHC/image_b6e0dc5.png)

### 移出节点
1. 队列中空闲的节点可移出队列，移出后可加入其他队列中。
2. 登录[百舸异构计算平台AIHC控制台](https://console.bce.baidu.com/aihc/resources)。  
3. 进入资源队列详情，选择**节点管理**，找到要封锁的节点，点击操作栏中的“移出”，确认后即可移出队列。  
![image.png](https://bce.bdstatic.com/doc/bce-doc/AIHC/image_5ba5fe1.png)
