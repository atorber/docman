## 概述
托管资源池网络是承载托管资源池计算节点和容器的网络连接，支持用户自定义配置CIDR网段，同时承担了该网段与用户指定VPC互通的职责，因此配置的CIDR网段不能与互通VPC网段有冲突，用户可以根据自己的实际情况进行选择。  

## 操作步骤
1. 登录[百舸异构计算平台AIHC控制台](https://console.bce.baidu.com/aihc/resources)。  
2. 在左侧菜单栏选择托管资源池，进入托管资源池页面，选择**网络**，点击**创建**。  
![image.png](https://bce.bdstatic.com/doc/bce-doc/AIHC/image_5f2ee05.png)
3. 在创建网络页面，根据实际需求完成相关参数配置。

| 配置项 | 必填/选填 | 配置说明 |
| --- | --- | --- |
| 名称 | 必填 | 托管资源池网络名称，支持大小写字母、数字、中文以及-_ /.特殊字符，必须以字母开头，长度1-65。 |
| 互通VPC | 必填 | 百舸打通当前用户账号的VPC私有网络 |
| 服务发布子网 | 必填 | 用于Tensorboard、推理服务发布使用。 |
| 节点数 | 选填 | 计划要加入到当前资源池的节点数，系统可根据节点数推荐百舸节点网段和容器网段 |
| 百舸节点网段 | 必填 | 百舸节点网段用于分配给百舸节点使用，百舸节点网段不能与VPC中已分配的子网冲突，可选网段：10.0.0.0/8-24、172.16.0.0/12-24、192.168.0.0/16-24。 |
| 百舸容器网段 | 必填 | 百舸容器网段用于分配给训练、推理等工作负载容器使用，百舸容器网段不能与百舸节点网段冲突，且不能与VPC中已分配的子网冲突，可选网段：10.0.0.0/8-16、 172.16.0.0/12-16、192.168.0.0/16。 |
![image.png](https://bce.bdstatic.com/doc/bce-doc/AIHC/image_68dd0fa.png)

4. 确认网络配置信息，然后单击提交。创建成功后，您可以在**网络**页面查看已创建的网络列表。
