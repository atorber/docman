  

## 简介

音频格式转换处理器



### 功能描述

* 支持音频格式转换（WAV、MP3、FLAC）
* 自动选择合适的编码器
* 通过extra_params支持自定义ffmpeg参数
* 支持本地文件, BOS地址存储



## 算子参数

### 输入

| 输入       | 含义                                  |
| ---------- | ------------------------------------- |
| input_col  | 包含输入音频路径的数组（支持本地路径) |
| output_col | 输出音频路径的数组（支持本地路径)     |

### 输出

| 输出           | 含义                                                   |
| -------------- | ------------------------------------------------------ |
| convert_result | 包含转换结果路径的数组，成功返回输出路径，失败返回None |



### 参数

| 参数名称      | 类型         | 默认值 | 描述                                                         |
| ------------- | ------------ | ------ | ------------------------------------------------------------ |
| output_format | str          | None   | 输出音频格式，仅支持 "wav", "mp3", "flac"，默认为 "wav"      |
| timeout       | int or None  | None   | ffmpeg 执行超时时间（秒），默认为 None（无超时）             |
| extra_params  | list or None | None   | 额外的 ffmpeg 参数列表，直接拼接到命令中 例如： - 音轨选择: ["-map", "0:a"] # 选择所有音轨 - 采样率: ["-ar", "44100"] - 比特率: ["-b:a", "192k"] # 适用于 MP3 - 压缩级别: ["-compression_level", "8"] # 适用于 FLAC |

## 调用示例

```
from __future__ import annotations

import os
import daft
from daft import col

from daft.aihc.common.udf import aihc_udf
from daft.aihc.functions.audio.audio_convert import AudioConvert

if __name__ == "__main__":
    if os.getenv("DAFT_RUNNER", "native") == "ray":
        import ray
        ray.init(dashboard_host="0.0.0.0", ignore_reinit_error=True)
        daft.set_runner_ray()
    daft.set_execution_config(actor_udf_ready_timeout=6000, min_cpu_per_task=0)
   
    samples = {
        "input_path": [
            "file:///local/sample_1.wav",
            "file:///mnt/pfs/sample_2.wav",
            "file:///mnt/bos/sample_3.wav",
        ]
        "output_path": [
            "file:///local/output/sample_1.map3",
            "file:///mnt/pfs/output/sample_2.mp3",
            "file:///mnt/bos/output/sample_3.mp3",
        ],
    }
    ds = daft.from_pydict(samples)
    constructor_kwargs = {
        "output_format": "mp3",
        "sample_rate": 44100,
        "audio_map": "auto",
        "extra_params": ["-c:a", "libmp3lame", "-b:a", "192k", "-q:a", "2"],
    }
    ds = ds.with_column(
        "convert_result",
        aihc_udf(
            AudioConvert,
            construct_args=constructor_kwargs,
            num_cpus=1,
            concurrency=1,
            batch_size=1,
        )(col("input_path"), col("output_path")),
    )
    ds.show()
```
