
## 概述

本文档介绍全托管资源池中资源队列负载类型的配置与使用方法，涵盖创建资源队列时配置负载类型、在资源队列列表中查看和筛选负载类型，以及在不同业务场景下选择可用资源队列。通过为资源队列配置负载类型，您可以按业务场景对队列进行管理，并在开发机、分布式训练、在线服务部署等页面中使用匹配的资源队列，实现资源的精细化管理。

## 前提条件

- 已登录百度智能云控制台，并可访问全托管资源池相关页面。
- 已具备 队列管理员、资源池管理员或 AIHCfullcontrol 权限。
- 已在目标地域创建可用的全托管资源池；如需在业务页面选择资源队列，请确保该地域下已存在可用资源队列。

## 负载类型使用说明

资源队列支持以下负载类型：

| 业务场景 | 可见队列规则 |
| --- | --- |
| 开发机 | 仅显示已配置 **开发机** 负载类型的资源队列 |
| 分布式训练 | 仅显示已配置 **分布式训练** 负载类型的资源队列 |
| 在线服务部署 | 仅显示已配置 **在线服务部署** 负载类型的资源队列 |

- 一个资源队列可以同时配置多个负载类型。
- 在资源队列提交前，您可以调整已选择的负载类型。
- 编辑资源队列时，可以新增负载类型；已经生效的负载类型不可直接删除。
- 负载类型变更仅影响后续新建、提交或部署操作，不影响已运行的开发机、分布式训练任务和在线服务部署实例。
- 功能上线前已存在的资源队列默认支持全部负载类型。

## 创建资源队列并配置负载类型

导航路径：**全托管资源池->资源队列**

### 创建资源队列时设置负载类型

在资源队列列表页，点击 **【创建资源队列】**，进入资源队列创建页面。

![创建资源队列页面](https://prd2md-online.bj.bcebos.com/v1/images/3cad7229-04da-4e89-96a3-7f52c72de6c4.png?authorization=bce-auth-v1%2FALTAKQw61w60kfkUunRcbQolTo%2F2026-03-13T11%3A09%3A05Z%2F-1%2Fhost%2Ff36ffd3f6de272b0fef83177ebaa94538d0a89307c23f7c1ca1700e314e652e2)


在创建页面填写基础信息，并在 **负载类型** 字段中选择资源队列支持的业务场景。

| 参数 | 必填 | 说明 |
| --- | --- | --- |
| 队列名称 | 是 | 输入自定义队列名称，用于区分不同资源队列。 |
| 资源池 | 是 | 选择当前地域下已创建的资源池。未选择资源池时，页面无法提交。 |
| 负载类型 | 是 | 选择资源队列支持的业务场景，可多选。可选项包括 **开发机**、**分布式训练**、**在线服务部署**。 |
| 备注 | 否 | 输入便于识别的说明信息。 |



### 查看负载类型列&编辑负载类型



进入资源队列列表页后，可在列表中查看新增的 **负载类型** 列，用于识别每个资源队列支持的业务场景。

## 在业务页面选择可用资源队列

### 场景入口

- 开发机：**开发机->创建实例**
- 分布式训练：**分布式训练->创建任务**
- 在线服务部署：**在线服务部署->部署服务**

### 使用规则

- 在业务页面打开资源队列选择控件后，系统仅展示支持当前业务场景的资源队列。
- 如果当前场景下没有可选资源队列，请返回 **全托管资源池->资源队列**，检查队列是否已配置对应的 **负载类型**。
- 如果一个资源队列同时配置了多个负载类型，则该资源队列可以在多个业务场景中复用。

### 变更影响说明

- 修改资源队列的负载类型后，后续新建的开发机、分布式训练任务和在线服务部署任务会按最新配置选择队列。
- 已经运行中的开发机、分布式训练任务和在线服务部署实例不受影响。
