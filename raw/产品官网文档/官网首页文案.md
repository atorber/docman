## 适用场景

- 版本发布前进行帮助文档一致性检查
- 需求评审前进行多角色风险预审
- 新功能上线时快速生成帮助文档草稿
- 需求探索阶段快速生成 PRD 初稿并持续迭代

## 使用流程

1. 在 Web 页面选择任务并配置参数  
2. 一键生成 Prompt  
3. 在本地 Agent 执行对应 Skill  
4. 返回 DocMan 查看报告与历史记录

## 信任与保障

- 可本地部署，数据留在团队可控环境
- 路径与输出规则明确，结果管理可追溯
- 支持健康检查与标准化接口，便于集成
