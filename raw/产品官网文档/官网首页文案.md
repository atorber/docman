## 核心价值（Value Proposition）

### 价值点 1：统一入口

在一个工作台内完成文档与 PRD 全流程任务，减少工具切换成本。

### 价值点 2：统一标准

内置结构化诊断维度与评审视角，让输出结果可对齐、可复用。

### 价值点 3：统一记录

最近记录聚合展示，支持按来源筛选与详情跳转，复盘更高效。

---

## 功能亮点（Feature Highlights）

### 帮助文档诊断

覆盖 18 个诊断维度，支持报告查看、问题定位、修复前后对比与在线修订。

### PRD 多视角评审

覆盖产品、研发、架构、测试、文档质量 5 大视角与 32 个维度，输出结构化评审结论。

### 帮助文档生成

基于 PRD 与控制台 URL 生成任务 Prompt，快速产出可落地的帮助文档内容。

### PRD 生成

基于需求描述、竞品链接与参考文档，生成高质量 PRD 任务 Prompt。

### 最近记录中心

统一聚合四类任务记录，支持来源筛选与一键查看详情，形成任务闭环。

## 适用场景（Use Cases）

- 版本发布前进行帮助文档一致性检查
- 需求评审前进行多角色风险预审
- 新功能上线时快速生成帮助文档草稿
- 需求探索阶段快速生成 PRD 初稿并持续迭代

## 使用流程（How It Works）

1. 在 Web 页面选择任务并配置参数  
2. 一键生成 Prompt  
3. 在本地 Agent 执行对应 Skill  
4. 返回 DocMan 查看报告与历史记录
