# 使用流程与导航

## 文档准备规范

- 帮助文档输入：放在 `raw/`
- PRD 相关输入：放在 `prd/`
- 产物目录：`report/`、`timeline/`、`new/`、`screenshots/`

## 标准使用流程

1. 进入目标页面并选择任务类型
2. 配置任务参数并生成 Prompt
3. 在本地 Agent 执行 Prompt
4. 回到工作台查看结果和历史记录

## 页面导航

- `/` 或 `/diagnose`：帮助文档诊断
- `/docgen`：帮助文档生成
- `/prd-review`：PRD 评审
- `/records`：最近记录聚合页
- `/prd-generate`：PRD 生成入口（实际跳转为 `/?nav=prdgen`）

## 记录跳转与分享

- 诊断与评审页面支持通过 URL 参数直达文档和记录
- 最近记录支持按来源筛选并一键查看详情
- 适用于团队内复盘、问题定位与协作同步
