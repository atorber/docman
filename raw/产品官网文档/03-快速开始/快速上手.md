# 快速上手

## 环境准备

- Node.js >= 18
- npm >= 8

## 3 分钟启动

### 1) 安装依赖

```bash
cd backend && npm install
cd ../web && npm install
```

### 2) 启动服务（推荐）

```bash
npm run dev
```

### 3) 打开页面

- 前端工作台：`http://localhost:3000`
- 后端健康检查：`http://localhost:3001/health`

## 常用启动命令

- 联合启动：`npm run dev`
- 联合启动（后端调试）：`npm run dev:debug`
- 仅后端：`npm run dev:backend`
- 仅前端：`npm run dev:web`

## 首次体验建议

1. 在 `raw/` 放入一篇待诊断文档
2. 在页面生成诊断 Prompt 并交给本地 Agent 执行
3. 回到“最近记录”查看报告与修复结果
