快速开始对百舸分布式训练、在线服务部署、开发机等进行了模版封装，涵盖LLM、AIGC、计算机视觉、等多种AI场景，帮助您零代码，高效快捷地部署、训练、开发开源大模型、具身智能模型等，从而快速上手百舸功能，验证业务需求。

本文带你通过快速开始快速上手理解使用百舸分布式训练、在线服务部署、开发机三个核心功能模块。

## 前提条件

1. 使用主账号开通百舸平台

2. 登录百舸控制台，购买资源池

## 模型部署

1. 登录百舸控制台，在左侧导航栏单击快速开始，搜索并找到对应模型选项卡，然后单击部署。

![image.png](https://bce.bdstatic.com/doc/bce-doc/AIHC/image_6f77702.png)

2. 配置部署参数。在部署配置页面已为您预置了默认参数，直接单击部署 > 确定。

![image.png](https://bce.bdstatic.com/doc/bce-doc/AIHC/image_b94248e.png)

点击确定后会跳转到在线服务部署列表，需要等待几分钟，当服务处于运行中时，代表部署成功。

3. 调用模型

查看调用信息。在服务列表页面，单击查看调用信息获取调用地址和Token。

![image.png](https://bce.bdstatic.com/doc/bce-doc/AIHC/image_f5ff74c.png)

体验模型服务。常用的调用方式如下：

调用示例

```bash
curl --location '<访问地址>/v1/chat/completions' \
--header 'Content-Type: application/json' \
--header 'Authorization: <TOKEN>' \
--data '{
    "model": "ERNIE-4.5-300B-A47B",
    "messages": [{"role": "system", "content": "你是一名天文学家,请回答用户提出的问题。"}, {"role": "user", "content": "人类是否能登上火星?"}, {"role": "assistant", "content": "目前来看,人类登上火星是完全可能的..."}],
    "max_tokens": 1024,
    "temperature": 0
}'
```

## 模型训练

1. 在左侧导航栏单击快速开始，搜索并找到RDT-1B选项卡，然后单击训练。

![image.png](https://bce.bdstatic.com/doc/bce-doc/AIHC/image_05ff709.png)

配置训练任务参数。只需配置如下关键参数，其他参数默认即可。

![image.png](https://bce.bdstatic.com/doc/bce-doc/AIHC/image_1484d68.png)

下表给出了不同模型默认参数和数据集的情况下，训练所依赖的最低配置：

|模型名称|训练方式|GPU资源|
|-|-|-|
|robotics-diffusion-transformer/rdt-1b|Pretrain|8卡，H800/A800等资源|
|robotics-diffusion-transformer/rdt-1b|Finetune|1-8卡，H800/A800等资源|

2. 点击确定后会跳转到分布式训练任务列表，训练任务进入创建中状态，需要等待几分钟，当处于运行中时开始训练。

查看训练任务并等待训练完成。在训练过程中，任务详情页面将展示任务日志、事件及指标监控，待训练执行完成后，模型将存储到设置的BOS目录中并自动注册到模型管理。


## 在开发机中打开

1. 在左侧导航栏单击快速开始，搜索并找到Isaac Lab选项卡，然后单击在开发机中打开。

配置启动参数。只需配置如下关键参数，其他参数默认即可。

![image.png](https://bce.bdstatic.com/doc/bce-doc/AIHC/image_6acf9f3.png)

2. 点击确定后会跳转到开发机列表，需要等待几分钟，当处于运行中时开发机可以正常使用。

![image.png](https://bce.bdstatic.com/doc/bce-doc/AIHC/image_5fac0d8.png)

3. 点击登录，可以登录开发机使用开发机内置的WebIDE和Notebook进行开发调试。

![image.png](https://bce.bdstatic.com/doc/bce-doc/AIHC/image_e76336d.png)
