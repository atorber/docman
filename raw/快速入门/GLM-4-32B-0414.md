GLM-4-32B-0414 是由智谱AI于2025年4月推出的轻量化开源大语言模型，参数量达 320 亿。其性能可与 OpenAI 的 GPT 系列和 DeepSeek 的 V3/R1 系列相媲美，并支持非常友好的本地部署特性。

该模型在工程代码、Artifact 生成、函数调用、基于搜索的问答以及报告生成等方面均取得了良好效果。尤其在多个基准测试中（如代码生成或特定问答任务），其表现可与 GPT-4o 和 DeepSeek-V3-0324（671B）等更大规模模型相媲美。

## API调用
1. 服务部署成功后，可在服务列表查看调用信息

2.调用示例
```
curl -X POST "<访问地址>/v1/chat/completions" \
-H "Content-Type: application/json" \
-H "Authorization: Bearer <TOKEN>" \
-d '{
  "model": "GLM-4-32B-0414",
  "messages": [{"role": "system", "content": "你是一名天文学家，请回答用户提出的问题。"}, {"role": "user", "content": "人类是否能登上火星?"}],
  "max_tokens": 1024,
  "temperature": 0.7
}'
```
