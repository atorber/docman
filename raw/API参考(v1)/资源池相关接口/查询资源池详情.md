### 描述
获取百舸资源池列表的接口
### 请求结构
```bash
GET /api/v1/resourcepools/{resourcePoolId}
Host:aihc.bj.baidubce.com
Authorization:authorization string
ContentType: application/json
```
### 请求头域
除公共头域外，无其它特殊头域。
### 请求参数
| 参数名称      | 类型   | 是否必须 | 参数位置 | 说明                   |
|--------------|--------|----------|----------|------------------------|
| resourcePoolId | String | 是       | Path参数 | 标识资源池的唯一标识符 |
### 返回头域
除公共头域，无其它特殊头域。
### 返回参数
| 参数名称      | 类型   | 是否必须 | 说明                     |
|-----------|------|------|------------------------|
| requestId | String | 是   | 请求 ID, 用于标识请求的唯一ID |
| result    | [ResourcePoolInfoResult](AIHC/API参考(v1)/附录.md#ResourcePoolInfoResult)| 是   | 请求返回的结果对象 |
