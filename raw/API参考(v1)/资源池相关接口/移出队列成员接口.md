### 描述
移出百舸资源池队列成员
### 请求结构
```
POST /api/v1/resourcepools/{resourcePoolId}/queue/{resourcePoolQueueId}/removeUsers
Host:aihc.bj.baidubce.com
Authorization:authorization string
ContentType: application/json
```
### 请求头域
除公共头域外，无其它特殊头域。
### 请求参数
| 参数名称      | 类型   | 是否必须 | 参数位置 | 说明                   |
|--------------|--------|----------|----------|------------------------|
| resourcePoolId | String | 是       | Path参数 | 标识资源池的唯一标识符 |
| resourcePoolQueueId | String | 是       | Path参数 | 标识资源池队列的唯一标识符 |
| users | Array of String | 否       | Body参数 | 队列成员的用户ID |
| groups | Array of String | 否       | Body参数 | 队列成员的用户组ID|
| managerGroups | Array of String | 否       | Body参数 | 队列管理员的用户组ID|
| managerUsers | Array of String | 否       | Body参数 | 队列管理员的用户ID|


### 返回头域
除公共头域，无其它特殊头域。
### 返回参数
| 参数名称      | 类型   | 是否必须 | 说明                     |
|-----------|------|------|------------------------|
| requestId | String | 是   | 请求 ID, 用于标识请求的唯一ID |
| result    | bool| 是   | 是否添加成功 |
### 请求示例
```json
{
    "groups": [
    ],
    "managerGroups": [
    ],
    "managerUsers": [
        "da7875b3b0b945cfb4cff480c50da834"
    ],
    "users": [
        "f0ddefc9c223486caf319836e8df8d73"
    ]
}
```

### 返回示例
```json
{
    "requestId": "99afcd1e-4b5d-4793-a693-079accf42bad",
    "result": true
}
```
