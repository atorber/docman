### 描述
获取百舸资源池管理员列表
### 请求结构
```bash
GET /api/v1/resourcepools/{resourcePoolId}/users
Host:aihc.bj.baidubce.com
Authorization:authorization string
ContentType: application/json
```
### 请求头域
除公共头域外，无其它特殊头域。
### 请求参数
| 参数名称      | 类型   | 是否必须 | 参数位置 | 说明                   |
|--------------|--------|----------|----------|------------------------|
| resourcePoolId | String | 是       | Path参数 | 标识资源池的唯一标识符 |


### 返回头域
除公共头域，无其它特殊头域。
### 返回参数
| 参数名称      | 类型   | 是否必须 | 说明                     |
|-----------|------|------|------------------------|
| requestId | String | 是   | 请求 ID, 用于标识请求的唯一ID |
| result    | [ResourcePoolUsers](AIHC/API参考(v1)/附录.md#ResourcePoolUsers)| 是   | 请求成功时的响应结果 |

### 返回示例
```json
{
    "requestId": "3ff916b1-9ed2-4154-bd29-7da87f32e592",
    "result": {
        "users": [
            "f0ddefc9c223486caf319836e8df8d73"
        ]
    }
}
```
