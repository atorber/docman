  
## 概述

全托管资源池是百舸提供的免运维资源池。在全托管资源池下，用户无需关注资源池和算力资源的运维管理，由平台保障运维稳定性。通常情况下，建议为企业内部的独立团队或项目分别创建独立的资源池，以实现团队间或项目间的资源隔离。本文档介绍如何创建全托管资源池。

## 前提条件

- 已登录 [百舸AI计算平台AIHC控制台](https://console.bce.baidu.com/aihc/serverless/resource/list)。
- 账号具备创建全托管资源池的权限，并已准备目标地域和可用区下可选的 VPC 网络、节点子网、容器子网和安全组；如无可用资源，可在创建页按界面提示新建。
- 提交订单前，请确认账户余额满足页面校验要求；若页面提示 **账户余额** 待确认，请先按提示完成充值或补足可用金额。

## 操作步骤
导航路径：**AIHC 控制台->全托管资源池->资源池**

### 步骤 1：进入创建全托管资源池页面

在 **【全托管资源池】** 页面确认当前为 **【资源池】** 页签后，单击 **【创建资源池】**，进入 **创建全托管资源池** 页面。  

![infoflow 2026-03-31 14-05-54.png](https://bce.bdstatic.com/doc/bce-doc/AIHC/infoflow%202026-03-31%2014-05-54_306af85.png)

### 步骤 2：填写资源池配置

在 **创建全托管资源池** 页面，根据实际业务需求完成基础配置、网络配置、节点配置和关联服务配置。

**基础配置**

| 参数 | 必填 | 说明 |
| --- | --- | --- |
| 资源池名称 | 是 | 托管资源池名称。支持大小写字母、数字、中文及 `-`、`_`、`/`、`.` 等字符，必须以字母开头，长度为 1-65 个字符。 |
| 备注 | 否 | 按需填写资源池备注信息。 |
| 地域 | 是 | 按需选择资源池所属地域。 |
| 可用区 | 是 | 按需选择资源池所属可用区。 |

![image.png](https://bce.bdstatic.com/doc/bce-doc/AIHC/image_96b1613.png)


**节点配置**

| 参数 | 必填 | 说明 |
| --- | --- | --- |
| 付费方式 | 是 | 支持包年包月和按量付费，按需选择。 |
| 节点规格 | 是 | 选择资源池中需要创建的节点规格。 |
| 节点数量 | 是 | 填写资源池中需要创建的节点数量。 |
| 购买时长 | 否 | 选择包年包月时配置购买时长。 |
| 自动续费 | 否 | 选择包年包月时可配置自动续费。开通后，系统会在实例到期前按规则自动扣款续费，并通过短信和邮件提醒续费结果。 |
| 续费周期 | 否 | 开通自动续费后，按需选择按年或按月的续费周期及周期数量。 |

![infoflow 2026-03-31 15-34-07.png](https://bce.bdstatic.com/doc/bce-doc/AIHC/infoflow%202026-03-31%2015-34-07_d85fb09.png)

网络配置

| 参数 | 必填 | 说明 |
| --- | --- | --- |
| 资源池网络 | 是 | 选择资源池中节点使用的VPC网络，您可以在当前可用区新建VPC。 |
| 节点子网 | 是 | 资源池创建的节点将从节点子网中分配 IP 地址，您可以新建子网。 |
| 容器子网 | 是 | 资源池创建的Pod将从容器子网中分配 IP 地址，支持多选，您可以新建子网。 |
| 安全组 | 是 | 资源池节点使用的安全组，可选择普通安全组或企业安全组。 |

注：平台预留网段为11.0.0.0/12，资源池网络和容器子网不能与该网段冲突。

![infoflow 2026-03-31 15-36-27.png](https://bce.bdstatic.com/doc/bce-doc/AIHC/infoflow%202026-03-31%2015-36-27_db65be2.png)


**关联服务**

| 参数 | 必填 | 说明 |
| --- | --- | --- |
| 存储配置 | 否 | 选择需要挂载的 PFS 实例或挂载服务。创建资源池后，系统会自动将所选 PFS 挂载到资源池节点。 |
| 监控配置（CPROM） | 否 | 选择资源池监控指标推送的 CPROM 实例，用于保存监控数据并展示可观测大盘。 |

如无额外需求，**存储配置** 和 **监控配置（CPROM）** 可保持未选择状态，创建完资源池后有需求再关联绑定即可。

![PFS服务.png](https://bce.bdstatic.com/doc/bce-doc/AIHC/PFS%E6%9C%8D%E5%8A%A1_c024bfb.png)

### 步骤 3：确认订单并提交

> **注意：** 提交后将根据所选节点规格、数量和付费方式产生费用，请在提交前确认配置信息。

单击 **【确认订单】**，核对节点规格、数量、地域、可用区和费用信息。页面会展示与所选配置对应的订单摘要和参考价格。

在 **前置检查** 区域等待检查完成；若 **账户余额** 项显示待确认，请先确保账户可用金额大于 ￥100，或按页面提示完成充值。检查满足提交条件后，勾选 **《百度智能云线上订购协议》**，单击 **【提交】**。

创建成功后，您可以在 **【全托管资源池】** 页面查看已创建的资源池状态。
