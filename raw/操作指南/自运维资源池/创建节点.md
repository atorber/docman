

## 概述

**节点**是百度百舸平台中的重要组成部分，在百度百舸平台中，节点可以看作是计算、存储和容器等服务的一个集合，用户可以根据实际需求选择合适的节点来执行AI任务。同时，百度百舸平台还提供了丰富的API和工具，方便用户对节点进行管理和监控，以及对AI任务进行调度和优化。    
本文介绍如何查看节点详情、添加和移出节点，有助于您更为全面地了解并节点队列。

## 操作步骤

1. 登录[百舸AI计算平台AIHC控制台](https://console.bce.baidu.com/aihc/resources)。 
2. 在左侧菜单栏选择**资源池**，找到您想要查看的资源池名称，单击ID。
3. 单击**节点管理**页面，点击**添加节点**。

![image.png](https://bce.bdstatic.com/doc/bce-doc/AIHC/image_adcc8e2.png)

4. 在添加节点页面，根据实际需求完成相关参数配置。


| 配置项 | 必填/选填 | 配置说明 |
| --- | --- | --- |
| 节点来源 | 必填 | 新建节点或使用已有服务器，使用已有服务器支持选择已有BCC、EBC、BBC、HPAS节点 |
| 付费方式 | 必填 | 预付费或后付费 |
| 可用区 | 必填 | 当前资源池支持的容器子网所在的可用区，如可用区不可选，请先在资源池详情中添加对应可用区的容器子网 |
| 子网 | 必填 | 选择节点所在子网 |
| 节点选择 | 必填 | 选择要加入资源池中的节点套餐规格 |
| 节点数量 | 必填 | 本次加入资源池的节点数量 |
| 登录凭证 | 必填 | 选择登录节点的凭证，支持密钥对、用户自定义密码和创建后设置 |
| 封锁节点 | 选填 | 勾选时，本次加入资源池的节点默认开启封锁，新的任务不会调度到该节点上，通常在新节点加入资源池时开启，新节点检查符合预期后可关闭封锁。 |

![image.png](https://bce.bdstatic.com/doc/bce-doc/AIHC/image_dafd6ec.png)

5. 确认节点配置信息，然后单击提交。创建成功后，您可以在**节点管理**页面点击**节点名称**查看详情。
