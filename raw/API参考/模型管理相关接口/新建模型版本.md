### **接口描述**
新建模型版本。

### **权限说明**
鉴权认证机制的详细内容请参见[鉴权认证](https://cloud.baidu.com/doc/Reference/s/Njwvz1wot)。

### **请求结构**
```bash
POST ?action=CreateModelVersion&modelId=xxx
HTTP/1.1
Host: aihc.bj.baidubce.com
Authorization: authorization string
version: v2
```
### **请求头域**
除公共头域外，无其它特殊头域。

### **请求参数**
|参数名称|类型|是否必选|参数位置|参数描述|
|-|-|-|-|-|
|modelId|String|是|Query参数|模型ID|
|storageBucket|String|是|Body参数|模型存储的BOS桶|
|storagePath|String|是|Body参数|BOS桶中的存储路径|
|description|String|否|Body参数|描述|
|source|String|是|Body参数|该版本模型的来源UserUpload：用户上传|
|modelMetrics|String|否|Body参数|模型指标，JSON格式。格式如下，其中 Metrics、Dataset 中的内容用户可自定义填写：{"Results":[{"Metrics":{"loss":2.13,"lr":0.0005},"Dataset":{"DatasetId":"ds-xxx"}}]}|

### **响应头域**
除公共头域外，无其它特殊头域。

### **响应参数**
|参数名称|类型|描述|
|-|-|-|
|id|String|模型版本ID|

### **请求示例**
```json
{
    "storageBucket": "bucket2",
    "storagePath": "/path/to/version2",
    "description": "new version",
    "source": "UserUpload",
    "modelMetrics": "{\"Results\":[{\"Metrics\":{\"loss\":2.13,\"lr\":0.0005},\"Dataset\":{\"DatasetId\":\"ds-xxx\"}}]}"
}
```
### **响应示例**
```json
{
    "requestId": "6a85d6f8-444a-49ac-b3c5-b19e7adf6916",
    "id": "mo-FJ9ycYDq"
}
```
