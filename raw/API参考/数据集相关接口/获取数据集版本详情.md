### **接口描述**
获取数据集版本详情。

### **权限说明**
鉴权认证机制的详细内容请参见[鉴权认证](https://cloud.baidu.com/doc/Reference/s/Njwvz1wot)。

### **请求结构**
```bash
GET ?action=DescribeDatasetVersion&datasetId=xxx&versionId=xxx
HTTP/1.1
Host: aihc.bj.baidubce.com
Authorization: authorization string
version: v2
```
### **请求头域**
除公共头域外，无其它特殊头域。

### **请求参数**
|参数名称|类型|是否必选|参数位置|参数描述|
|-|-|-|-|-|
|datasetId|String|是|Query参数|数据集ID|
|versionId|String|是|Query参数|数据集版本ID|

### **响应头域**
除公共头域外，无其它特殊头域。

### **响应参数**
|参数名称|类型|描述|
|-|-|-|
|id|String|数据集ID|
|name|String|数据集名称|
|storageType|String|存储类型PFS：并行存储PFSBOS：对象存储BOS|
|storageInstance|String|存储实例|
|importFormat|String|导入格式FILE：文件FOLDER：文件夹|
|description|String|描述|
|owner|String|所有者|
|ownerName|String|所有者名称|
|visibilityScope|String|可见范围ALL_PEOPLE：所有人可见ONLY_OWNER：仅所有者可读写USER_GROUP：指定范围可用|
|visibilityUser|List<[PermissionEntry](AIHC/API参考/数据结构.md#PermissionEntry)\>|用户权限列表[{"id": "xxx", "name": "xxx", "permission": "r"}, {"id": "yyy", "name": "xxx", "permission": "rw"}]仅对管理员或Owner返回此属性|
|visibilityGroup|List<[PermissionEntry](AIHC/API参考/数据结构.md#PermissionEntry)\>|用户组权限列表[{"id": "xxx", "name": "xxx", "permission": "r"}, {"id": "yyy", "name": "xxx", "permission": "rw"}]仅对管理员或Owner返回此属性|
|permission|String|当前用户拥有的读写权限：r：只读rw：读写|
|versionEntry|[DatasetVersionEntry](AIHC/API参考/数据结构.md#DatasetVersionEntry)|指定版本相关信息|
|createdAt|String|创建时间|
|updatedAt|String|更新时间|
