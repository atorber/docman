### **接口描述**
获取数据集版本列表。

### **权限说明**
鉴权认证机制的详细内容请参见[鉴权认证](https://cloud.baidu.com/doc/Reference/s/Njwvz1wot)。

### **请求结构**
```bash
GET ?action=DescribeDatasetVersions&datasetId=xxx&pageNumber=xxx&pageSize=xxx
HTTP/1.1
Host: aihc.bj.baidubce.com
Authorization: authorization string
version: v2
```
### **请求头域**
除公共头域外，无其它特殊头域。

### **请求参数**
|参数名称|类型|是否必选|参数位置|参数描述|
|-|-|-|-|-|
|datasetId|String|是|Query参数|数据集ID|
|pageNumber|Integer|否|Query参数|请求分页参数，表示第几页，默认值为1|
|pageSize|Integer|否|Query参数|单页结果数，不传递该参数默认返回全部|

### **响应头域**
除公共头域外，无其它特殊头域。

### **响应参数**
|参数名称|类型|描述|
|-|-|-|
|totalCount|Integer|数据集版本总数|
|versions|List<[DatasetVersionEntry](AIHC/API参考/数据结构.md#DatasetVersionEntry)\>|数据集版本列表|

### **请求示例**
无。
