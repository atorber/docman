
### 接口描述
制作开发机镜像，只有开发机状态为运行中（status 为 3）的时候才能制作镜像。

### 请求结构
```
POST ?action=CreateDevInstanceImagePackJob
HTTP/1.1
Host:aihc.bj.baidubce.com
version: v2
Authorization:authorization string
{
    "imageName": imageName,
    "imageTag": imageTag,
    "registry": registry,
    "namespace": namespace,
    "username": username,
    "password": password,
    "devInstanceId": devInstanceId
}
```
### 请求头域
除公共头域外，无其它特殊头域。

### 请求参数
|参数名|参数类型|是否必须|参数位置|参数说明|
|-|-|-|-|-|
|imageName|string|是|Body 参数|镜像名称|
|imageTag|string|是|Body 参数|镜像 tag|
|registry|string|是|Body 参数|镜像仓库|
|namespace|string|是|Body 参数|镜像仓库命名空间|
|username|string|是|Body 参数|镜像仓库用户名|
|password|string|是|Body 参数|镜像仓库密码|
|devInstanceId|string|是|Body 参数|操作对象|

### 返回头域
除公共头域，无其它特殊头域。

### 返回参数
|参数名称|类型|说明|
|-|-|-|
|requestId|string|请求唯一标识|
|imagePackJobId|string|镜像制作任务ID|
