### 接口描述
可以通过此接口开启或者关闭开发机定时关机。只有运行中（status 为 3）的开发机允许定时关机。

### 请求结构
```
POST ?action=TimedStopDevInstance
HTTP/1.1
Host:aihc.bj.baidubce.com
version: v2
Authorization:authorization string
{
    "devInstanceId": devInstanceId,
    "delaySec": delaySec,
    "enable": enable
}
```
### 请求头域
除公共头域外，无其它特殊头域。

### 请求参数
|参数名|参数类型|是否必须|参数位置|参数说明|
|-|-|-|-|-|
|devInstanceId|String|是|Body参数|操作对象|
|delaySec|Number|否|Body参数|定时关机时间（单位秒，当 enable 为 true 时需大于 3600 秒）|
|enable|Bool|否|Body参数|开启或关闭定时关机|

### 返回头域
除公共头域，无其它特殊头域。

### 返回参数
|参数名称|类型|说明|
|-|-|-|
|requestId|string|请求唯一标识|
|devInstanceId|string|开发机ID|

### 请求示例
```
POST ?action=TimedStopDevInstance
HTTP/1.1
Host:aihc.bj.baidubce.com
version: v2
Authorization: bce-auth-v1/f81d3b34e48048fbb2634dc7882d7e21/2024-08-11T04:17:29Z/3600/host/74c506f68c65e26c633bfa104c863fffac5190fdec1ec24b7c03eb5d67d2e1de
{
    "devInstanceId": "dv-f458ede5",
    "delaySec": 3600,
    "enable": true
}
```
### 返回示例
```
{
    "devInstanceId": "dv-f458ede5",
    "requestId": "daa778cc-3625-gray-cce0-bcecanarytag"
}
```
