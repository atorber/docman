### 描述
查询队列节点列表，目前支持查询托管资源节点。

### 请求结构
```
GET ?action=DescribeNodes&resourcePoolId=xxx&pageSize=10&pageNo=2
Host:aihc.bj.baidubce.com
Authorization:authorization string
ContentType: application/json
version: v2
```
### 请求头域
除公共头域外，无其它特殊头域。

### 请求参数
|参数名称|类型|是否必须|说明|
|-|-|-|-|
|resourcePoolId|String|是|资源池名称|
|instanceId|String|否|实例ID|
|orderBy|String|否|排序字段：支持CreatedAt|
|order|String|否|排序方式，升序：asc，降序：desc|
|pageSize|Number|否|分页大小：默认为 10|
|pageNo|Number|否|页号：默认为 1|

### 返回头域
除公共头域外，无其他特殊头域。

### 返回参数
|参数名称|类型|说明|
|-|-|-|
|nodes|List<[Node](AIHC/API参考/数据结构.md#Node)>|节点列表|
|requestId|String|请求ID|
