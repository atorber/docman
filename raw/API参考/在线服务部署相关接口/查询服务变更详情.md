### 接口描述
查询服务变更详情

### 请求结构
```bash
GET ?action=DescribeServiceChangelog&changeId=changeId
HTTP/1.1
Host:aihc.bj.baidubce.com
version: v2
Authorization:authorization string 
```
### 请求头域
除公共头域外，无其它特殊头域。

### 请求参数
|参数名|参数类型|是否必须|参数位置|参数说明|
|-|-|-|-|-|
|changeId|String|是|Query参数|变更 ID|

### 返回头域
除公共头域，无其它特殊头域。

### 返回参数
|参数名|参数类型|参数说明|
|-|-|-|
|changeId|String|变更ID|
|prev|String|前一条变更记录|
|changeType|Integer|变更类型，1:创建 2:更新 3:扩缩容 4:流量接入变更|
|description|String|变更简短描述|
|creator|String|操作人|
|createdAt|Integer|变更创建时间|
|requestId|String|请求唯一标识|
