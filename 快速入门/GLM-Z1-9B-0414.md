GLM-Z1-9B-0414 是由智谱AI于2025年4月推出的轻量化开源大语言模型，参数规模为90亿（9B）。该模型采用“冷启动 + 强化学习”，在数学推理和通用任务方面表现出色，其整体性能在所有同规模开源模型中名列前茅。尤其在资源受限的场景下，该模型在效率与效果之间实现了出色的平衡，为寻求轻量级部署的用户提供了强大选择。

## API调用
1. 服务部署成功后，可在服务列表查看调用信息

![image.png](https://bce.bdstatic.com/doc/bce-doc/AIHC/image_bee5749.png)

2.调用示例
```
curl -X POST "<访问地址>/v1/chat/completions" \
-H "Content-Type: application/json" \
-H "Authorization: Bearer <TOKEN>" \
-d '{
  "model": "GLM-Z1-9B-0414",
  "messages": [{"role": "system", "content": "你是一名天文学家，请回答用户提出的问题。"}, {"role": "user", "content": "人类是否能登上火星?"}],
  "max_tokens": 1024,
  "temperature": 0.7
}'
```
