Llama2-7B 是 Meta 公司发布的 Llama 2 系列大语言模型中最小的一个版本，拥有 70亿 参数。尽管是系列中规模最小的模型，但它是一个功能强大且高效的模型，在性能、速度和硬件需求之间取得了出色的平衡。
## 模型基本特性

* 类型：自回归生成式文本大语言模型

* 参数规模：70亿

* 上下文长度：4096 Token

* 架构：基于 Transformer 架构，并进行了优化。它没有使用 Llama 2 70B 模型中的分组查询注意力，而是标准的多头注意力机制。

## 优势与特点

* 出色的性能/效率比： 在众多学术基准测试（如常识推理、代码生成等）上，其性能远超同规模的其他开源模型（如 MPT-7B, Falcon-7B）。

* 相对较低的硬件需求： 经过量化后（如使用4-bit或8-bit精度），它可以在消费级显卡（如 RTX 3090/4090， 甚至更低的 GPU）上流畅运行，这使得个人开发和研究者能够轻松部署和使用。

* 强大的对话能力： 其 Chat 版本在帮助性和安全性上经过了严格对齐，能够生成高质量、上下文相关的对话回复。

* 强大的生态系统： 由于其流行度和开源特性，Llama 2 7B 拥有极其丰富的衍生模型和工具支持（例如，与 LangChain 的集成、各种量化版本的 GGUF 文件等），社区活跃，易于找到相关资源和解决方案。

## API调用
1.服务部署成功后，可在服务列表查看调用信息
![image.png](https://bce.bdstatic.com/doc/bce-doc/AIHC/image_f08e456.png)

2.调用示例
```
curl --location '<访问地址>/v1/completions' \
--header 'Content-Type: application/json' \
--header 'Authorization: <TOKEN>' \
--data '{
    "model": "Llama2-7B",
    "prompt": ["鸡兔同笼，头共20个，足共62只，求鸡与兔各有多少只？"],
    "max_tokens": 2000,
    "temperature": 1
}'
```
