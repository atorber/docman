## 模型介绍

Step-3.5-Flash 是阶跃星辰最强大的开源基础模型，专为以卓越效率提供前沿推理与智能体（agentic）能力而设计。该模型基于稀疏的混合专家（MoE）架构，每处理一个 token 仅激活其 1960 亿参数中的 110 亿个。这种“智能密度”使其推理深度可媲美顶级闭源模型，同时保持实时交互所需的敏捷性。

## 核心能力

* **高速深度推理：**聊天机器人侧重于阅读，而智能体必须快速推理。得益于 3-way Multi-Token（MTP-3）预测技术，Step-3.5-Flash 在典型使用场景下生成速度可达 100–300 tok/s（在单流编码任务中峰值可达 350 tok/s）。这使其能够执行复杂的多步推理链，并保持即时响应。

* **面向代码与智能体的稳健引擎：**Step-3.5-Flash 专为智能体任务打造，集成了可扩展的强化学习（RL）框架，推动持续的自我改进。它在 SWE-bench Verified 上达到 74.4%，在 Terminal-Bench 2.0 上达到 51.0%，证明了其在处理复杂、长周期任务时具备极强的稳定性。

* **高效的长上下文支持：**该模型采用 3:1 的滑动窗口注意力（Sliding Window Attention, SWA）比例——即每层全注意力层搭配三层 SWA 层，从而以较低成本支持 256K 上下文窗口。这种混合方法在大规模数据集或长代码库上保持一致性能的同时，显著降低了标准长上下文模型常见的计算开销。

* **便捷的本地部署：**为提升可访问性，Step-3.5-Flash 将顶尖智能能力带入本地环境。它可在高端消费级硬件（如 Mac Studio M4 Max、NVIDIA DGX Spark）上安全运行，在保障数据隐私的同时不牺牲性能。

## 架构细节

Step-3.5-Flash 基于稀疏的混合专家（MoE）Transformer 架构构建，针对推理过程中的高吞吐量和低显存占用进行了优化。

* **技术规格**

    * **主干网络：**45 层 Transformer（隐藏层维度 4,096）

    * **上下文窗口：**256K
    
    * **词表大小：**128,896 个 token

    * **参数量：**总参数量196.81B（196B 主干 + 0.81B 输出头），激活参数量约 11B（每 token 生成）

* **混合专家（MoE）路由机制**

    与传统稠密模型不同，Step-3.5-Flash 采用细粒度路由策略以最大化效率：

    * **细粒度专家：**每层包含 288 个路由专家 + 1 个共享专家（始终激活）。
    
    * **稀疏激活：**每个 token 仅选择 Top-8 个专家。
    
    * **效果：**模型保留了 196B 参数模型的“记忆”能力，但以 11B 模型的速度执行。

* **多 Token 预测（MTP）**

    为提升推理速度，使用了一个专用的 MTP Head，包含滑动窗口注意力机制和稠密前馈网络（FFN）。该模块可在单次前向传播中同时预测 4 个 token，在不降低质量的前提下显著加速推理。

## API调用

1. 服务部署成功后，可在服务列表查看调用信息

![image.png](https://bce.bdstatic.com/doc/bce-doc/AIHC/image_bee5749.png)

2.调用示例

```
curl -X POST "<访问地址>/v1/chat/completions" \
-H "Content-Type: application/json" \
-H "Authorization: Bearer <TOKEN>" \
-d '{
    "model": "Step-3.5-Flash",
    "messages": [{"role": "system", "content": "你是一名天文学家,请回答用户提出的问题。"}, {"role": "user", "content": "人类是否能登上火星?"}],
    "max_tokens": 1024,
    "temperature": 0.6
}'
```
