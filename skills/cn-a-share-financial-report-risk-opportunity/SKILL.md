# A股财报风险/机会分析（外部对比驱动）

## 适用场景

- 用户提供年报/半年报（PDF/MD/文本），希望快速识别财报中的**风险点**与**机会点**
- 用户希望输出“简洁研究报告”，并以**要点清单**突出结论
- 默认需要：
  - 同一公司最近 **3 期**趋势对比
  - 同行业可比公司 **分位**对比
- 强约束：每次运行都要发起外部对比请求并用于论证（来自：东方财富优先；失败则降置信度）

## 输入参数（prompt 提供）

| 参数 | 必填 | 说明 |
| --- | --- | --- |
| 原始文档路径 | ✅ | 相对于项目根目录的 `finance_reports/` 目录路径，如 `财报/600519_2024年报.pdf` |
| 股票代码/简称 | ✅ | 如 `600519` 或 `贵州茅台`（用于外部数据源检索） |
| 报告类型 | ✅ | `年报` / `半年报`（可支持季报但默认按年报/半年报） |
| 报告期 | ✅ | 如 `2024年` / `2025H1` |
| 时间戳 | ✅ | `YYYYMMDD_HHMMSS`（输出文件命名用） |
| 外部数据优先级 | ❌ | 默认 `东方财富`（可空；本 skill 固定以东方财富优先） |
| 关注偏好 | ❌ | 如 `偏风险/偏机会/偏现金流/偏盈利质量`（可空） |

## 输出配置（固定默认）

- 主报告输出：`report/finance/cn-a-share/risk-opportunity/{股票代码}/{报告期}_{时间戳}_report.md`
- 过程记录（可选但建议）：`timeline/finance/cn-a-share/risk-opportunity/{股票代码}/{报告期}_{时间戳}_timeline.json`

## 执行流程（每次都要按顺序做）

### Step 1：读取并抽取“证据”

从用户提供的年报/半年报文档（PDF/MD/文本）中抽取并结构化：

1. 定量证据（用于趋势与归因）
   - 利润表核心：营收、归母净利、扣非净利（如有）、毛利率/净利率（如有）
   - 现金流核心：经营活动现金流净额（如有）
   - 资产负债关键：应收账款、存货、短期/长期负债、现金及等价物（如有）
2. 定性证据（用于“为什么有风险/机会”）
   - 附注/MD&A/风险提示/重大事项说明中与下列主题相关的段落：
     减值、诉讼仲裁、担保/或有负债、回购/业绩补偿、收入/成本变动原因、经营模式变化
3. 口径线索
   - 是否存在会计政策变更、合并范围变化、一致性说明、一次性因素等

输出内部结构（仅供本次推理）：
`{指标 -> 数值与来源位置, 段落 -> 主题标签}`（不要求外部展示）

### Step 2：外部对比三件套（硬约束：每次都必须请求并用于论证）

本 step 在产出任何“高置信度”结论前必须完成。

#### 2.1 公司最近 3 期趋势对比（必须）

从外部数据源获取最近 3 个匹配频率的报告期数据（年报→年报；半年报→半年报）：

- 营收、归母净利、扣非净利（若外部可得）
- 毛利率/净利率（若外部可得）
- 经营活动现金流净额（若外部可得）
- 应收/存货（若外部可得）
产出：每个关键指标的“方向 + 是否出现拐点/背离”的可引用结论。

#### 2.2 行业分位对比（必须）

从外部获取行业分类口径（必须写明行业口径来源/名称），计算同行业可比公司分位：

- 对比指标按可得子集选择（建议优先盈利能力与现金兑现相关指标）
  - 盈利：毛利率、净利率、ROE/ROA（若外部可得）
  - 风险结构：资产负债率、短债压力类（若外部可得）
  - 现金兑现：经营现金流与利润匹配相关指标（若外部可得）
产出：关键指标在行业内的分位（percentile/rank），并给出“偏离方向”。

#### 2.3 重大事件外部佐证对照（必须“尝试请求”，失败则降置信度）

外部请求并核验与文档主题相关的事件摘要：

- 减值计提、诉讼仲裁、担保/或有负债、回购/业绩补偿、重大经营变化等
产出：事件是否与文档风险/机会表述一致；若无法核验则记录缺失项。

### Step 3：置信度判定与降级规则（强制执行）

对每条风险/机会分别给出置信度：`高 / 中 / 低`

- 高：文档证据明确 + 最近3期趋势一致/有拐点 + 行业分位偏离方向与趋势一致 + 事件核验一致
- 中：文档证据明确 + 趋势一致 + 但行业分位或事件核验缺失/不完整
- 低：外部对比三件套缺失较多，或无法对齐口径

降级规则（必须遵守）：

- 若行业分位对比缺失：所有引用行业分位的结论必须降为 `中/低`
- 若事件核验缺失：所有引用“为什么发生”的事件解释必须降为 `中/低`

### Step 4：风险点与机会点生成（默认 Top 5）

每条记录必须包含以下字段（用于前端展示/集成）：

- 主题（风险/机会具体是什么）
- 结论一句话
- 文档证据（数字与/或原文段落要点）
- 趋势证据（最近3期的方向、是否拐点/背离）
- 外部对比论证（行业分位偏离 + 事件是否匹配；缺失则写缺失项）
- 影响推演（对经营/现金流/偿债/盈利质量的可能传导，用 1-2 句）
- 待验证清单（下一期财报重点复查点）
- 置信度（高/中/低）

#### 风险主题库（命中优先）

- 盈利质量风险：净利与经营现金流背离；扣非长期偏弱；利润结构异常
- 应收/回款风险：应收增速显著高于收入；账龄恶化；坏账计提口径变化
- 存货与减值风险：存货占比抬升；周转变慢；跌价准备/减值迹象
- 商誉/无形资产减值风险：减值计提集中；资产结构变化
- 或有负债/诉讼仲裁/担保风险：附注风险提示增强 + 外部事件可对照
- 偿债/流动性风险（外部可得时）：短债压力上升、现金覆盖走弱

#### 机会主题库（命中优先）

- 现金兑现机会：利润改善能承接到经营现金流（趋势同向 + 行业不差）
- 成本/费用优化机会：费用率下降且与业务增长匹配（非一次性）
- 营运资本释放机会：应收/存货改善带来现金释放（需文档解释可持续性）
- 结构性机会：毛利率/利润率结构改善存在明确经营驱动
- 事件型修复机会：减值压力缓解或重大事项落地后指标修复（需外部事件佐证）

### Step 5：生成输出报告（简洁研究报告 + 要点清单）

## 报告模板（必须输出）

1. 一句话结论（风险主导/机会主导/两者并存 + 置信度一句话）
2. 关键指标速览（最近3期趋势，<=6项；每项写“方向/拐点/背离”）
3. 风险点（Top 5，按严重度降序）
4. 机会点（Top 5，按确定性降序）
5. 外部对比完成情况（必写）
   - 最近3期趋势对比：成功/失败 + 失败原因/缺失项
   - 行业分位对比：成功/失败 + 行业口径信息（或缺失原因）
   - 重大事件对照：成功/失败 + 缺失项
6. 数据口径与限制（口径变化、缺失字段、无法对齐项）
7. 免责声明（仅供研究分析，不构成投资建议）

## 异常处理

- 文档缺失或无法解析：直接提示用户提供可解析的 PDF/MD/文本并终止
- 外部对比三件套任一失败：仍输出报告，但必须降置信度并在“外部对比完成情况”中写明缺失项
