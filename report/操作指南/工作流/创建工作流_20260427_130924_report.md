### 默认配置内容验证

编辑页面中默认的YAML配置包含以下参数：

```yaml
taskTemplates:
  - name: custom-task-template
    type: CustomTask
    inputs:
      - name: queue_id      # 队列ID（需填写）
      - name: command       # 执行命令（需填写）
      - name: image_url     # 镜像地址（需填写）
      - name: cpu           # CPU配额
      - name: memory        # 内存配额
```

**结论**：文档"只需要修改queue参数"的描述不准确，用户实际需要填写多个参数才能运行工作流。
