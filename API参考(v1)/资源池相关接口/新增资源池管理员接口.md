### 描述
新增百舸资源池管理员
### 请求结构
```bash
POST /api/v1/resourcepools/{resourcePoolId}/addUsers
Host:aihc.bj.baidubce.com
Authorization:authorization string
ContentType: application/json
```
### 请求头域
除公共头域外，无其它特殊头域。
### 请求参数
| 参数名称      | 类型   | 是否必须 | 参数位置 | 说明                   |
|--------------|--------|----------|----------|------------------------|
| resourcePoolId | String | 是       | Path参数 | 标识资源池的唯一标识符 |
| users | Array of String | 否       | Body参数 | 用户ID |
| groups | Array of String | 否       | Body参数 | 用户组ID|


### 返回头域
除公共头域，无其它特殊头域。
### 返回参数
| 参数名称      | 类型   | 是否必须 | 说明                     |
|-----------|------|------|------------------------|
| requestId | String | 是   | 请求 ID, 用于标识请求的唯一ID |
| result    | bool| 是   | 是否添加成功 |
### 请求示例
```json
{
    "users": ["7137d425bf25412abbb3a8ac14b86004"],
    "groups": []
}
```

### 返回示例
```json
{
    "requestId": "99afcd1e-4b5d-4793-a693-079accf42bad",
    "result": true
}
```
