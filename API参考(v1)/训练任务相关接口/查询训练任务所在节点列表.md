### 描述

查询训练任务所在节点的名称

### 请求结构

```bash
GET /api/v1/aijobs/{jobID}/nodes
Host:aihc.bj.baidubce.com
Authorization:authorization string
ContentType: application/json    
```

### 请求头域

除公共头域外，无其它特殊头域。

### 请求参数

| 参数名称       | 类型   | 是否必须 | 参数位置   | 说明                   |
| -------------- | ------ | -------- | ---------- | ---------------------- |
| jobId          | String | 是       | Path 参数  | 训练任务ID             |
| resourcePoolId | String | 是       | Query 参数 | 标识资源池的唯一标识符 |

### 返回头域

除公共头域，无其它特殊头域。

### 返回参数

| 名称      | 类型                                          | 必选 | 说明     |
| --------- | --------------------------------------------- | ---- | -------- |
| requestId | String                                        | 是   | 请求ID   |
| result    | [NodeResult](AIHC/API参考(v1)/附录.md#NodeResult) | 是   | 返回结果 |

### 返回示例

```json
{
  "requestId": "string",
  "result": {
    "jobID": "",
    "nodeName": []
  }
}
```
